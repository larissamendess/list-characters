# Documentação - Lista de Personagens Start Wars - Smart Nx

Aplicação web desenvolvida em React.js para exibir e buscar personagens consumindo API externa da SWAPI.  


---

##  Tecnologias usadas

- React.js
- Typescript
- Ant Desing
- API SWAPI https://swapi.dev/
- React Hooks
- Vite 

---

##  Funcionalidades

- [x] Buscar personagens pelo nome através do input
- [x] Exibir lista de 10 personagens por página
- [x] É possivel navegar entre as paginas no componente de paginação tanto pelas setas, quanto pelos numeros das páginas.

---
##  Boas práticas

- [x] Componentização
- [x] Nomes descritivos e claros em variáveis, funções, e classes
- [x] Separação da chamada da API em uma pasta Service
- [x] Desestruturação de props 
- [x] Tratamento de erros, usando try catch

## Observações
Durante o desenvolvimento deste teste técnico, procurei implementar as principais funcionalidades e garantir o funcionamento da aplicação conforme os requisitos propostos.

Entretanto, algumas partes específicas não foram concluídas devido a limitações técnicas e de ambiente que encontrei durante o desenvolvimento:

Containerização (Docker): não foi possível concluir, pois não tenho permissão para instalar o Docker na máquina da empresa utilizada.

Testes unitários: encontrei incompatibilidades de versão ao configurar o ambiente de testes (Jest + React 17), e não consegui ajustar dentro do prazo mesmo tentando algumas alternativas.

Apesar disso, mantive a estrutura do projeto organizada e com base sólida para que essas partes possam ser facilmente adicionadas em um ambiente adequado.

Agradeço pela compreensão e pela oportunidade de demonstrar meu raciocínio técnico, organização e abordagem no desenvolvimento da solução!
